Phase 1: Setting up the Tools

1. Install Python (The Foundation)

    Windows: Go to Python.org and download the latest installer. Crucial step: When you run the installer, look at the very bottom of the first window and check the box that says "Add Python to PATH" before clicking install. If you miss this, nothing else will work.

    Mac: Modern Macs usually have Python pre-installed. If not, download the macOS installer from Python.org.

2. Open your Terminal / Command Prompt

    Windows: Click the Start button, type cmd, and hit Enter.

    Mac: Press Cmd + Space, type Terminal, and hit Enter.

3. Install pkg_resources
Type the following command into the terminal and press Enter:
pip install setuptools
Wait a few seconds for it to download and install. This puts the missing pkg_resources tool onto your computer.

4. Install spotDL
In the black box that opens, type this exact command and press Enter:
pip install spotdl
(Note: If you are on a Mac and it gives you an error, type pip3 install spotdl instead).

5. Install FFmpeg
spotDL needs a background tool called FFmpeg to physically convert the video streams into MP3 files. Windows has a built-in package manager that can download and install FFmpeg for you in about 10 seconds. Type this exact command into the black window and press Enter:
winget install ffmpeg
(Note: If it asks you to agree to any terms and conditions, just type Y and press Enter).
