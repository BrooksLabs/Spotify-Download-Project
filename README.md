# spotDL Python Guide & Helper

Interactive Python script that walks users through installing and using **[spotDL](https://github.com/spotDL/spotify-downloader)** — the popular Spotify → MP3 downloader.

## Features
- Step-by-step terminal guidance
- Checks & auto-installs spotDL + FFmpeg
- Folder selection helper
- Command generator with resume support
- Simple menu interface
